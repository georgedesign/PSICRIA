# PsiCria com Maven Wrapper

Mini site com backend em Java para criação de conteúdo com inteligência artificial voltado para psicólogos.

## Como rodar no Render

**Build Command:**
```
./mvnw clean install
```

**Start Command:**
```
java -jar target/psicria-1.0.0.jar
```
